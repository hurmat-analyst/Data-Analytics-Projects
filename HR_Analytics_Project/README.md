# HR Analytics Dashboard

## Project Overview

This HR Analytics Dashboard project focuses on analyzing employee attrition, workforce demographics, job satisfaction, and employee trends using Power BI. This Dashboard provides interactive insights that help understand workforce behavior and key HR matrix.

## Tools & Technologies
- Power BI
- Excel

---

## Key KPIs
- Total Employees
- Attrition Count
- Attrition Rate
- Active Employees
- Average Age
- Average Monthly Income

---

## Dashboard Features
- Employee Analysis by Age & Gender
- Attrition Analysis by Age
- Department wise Attrition
- Attrition Rate by Job Role
- Job Satifaction Rating Matrix
- Attrition Analysis by Education
- Interactive Slicers for:
  - Gender
  - Education
  - Business Travel
  - Marital Status
