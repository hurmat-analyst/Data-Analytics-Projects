# Bank Loan Analytics Dashboard

## Project Overview
This project focuses on analyzing bank loan applications, funded amounts, repayments, interest rate, and loan performance using Power BI.
The dashboard provides interactive insights into loan trends, customer behavior, and financial KPIs.

## Tools & Technologies
- Power BI
- Excel

---

## Key KPIs
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average DTI Ratio

---

## Dashboard Features
- Good Loan vs Bad Loan Analysis
- Monthly Loan Application Trends
- Loan Application by State
- Loan Application by Purpose
- Loan Application by Home Ownership
- Loan Status Analysis
- Interactive Filters & Slicers

---

## Project Workflow
1. Data Import from Excel
2. Data Modeling in Power BI
3. KPI Creation
4. Dashboard Development
5. Financial Insights Visualization
