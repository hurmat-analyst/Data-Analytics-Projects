# Customer Shopping Behavior Analysis

## Project Overview
This is an end-to-end Data Analytics project focused on customer shopping behavior analysis using Python, PostgreSQL, and Power BI.
This project includes:
- Data Cleaning & Exploratory Data Analysis(EDA)
- SQL-based Business Problem Solving
- Interactive Power BI Dashboard

---

## Tools & Technologies Used
- Python
- Pandas
- PostgreSQL
- pgAdmin4
- Power BI

---

## Project Workflow
1. Import and cleaned customer shopping data using python
2. Performed Exporatory Data Analysis(EDA)
3. Created PostgreSQL database and tables
4. Solved business questions using SQL queries
5. Built an interactive dashboard in Power BI

---

## Business Question Solved
- Top selling product categories
- Customer purchase behavior
- Sales trends analysis
- Revenue insights
- Customer spending patterns
