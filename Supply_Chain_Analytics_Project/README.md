# Supply Chain Analytics Dashboard

## Project Overview

This project focuses on inventory and supply chain management analysis using Power BI.

The dashboard provides insights into warehouse utilization, inventory levels, transportation costs, lead time analysis, and order status trends through interactive visualizations and KPIs.
---

## Tools & Technologies
- Power BI
- Excel

---

## Key KPIs
- Warehouse Utilizations
- Days Sales of Inventory
- Inventory Turnover Ratio

---

## Slicers
- Region
- Category

---

## Dashboard Features
- Transportation Cost Analysis by Region & Category
- Units Sold Trend Analysis
- Lead Time Analysis by Category
- Backorder Analysis by Order Status
- Inventory Level Analysis
- Interactive Filters & Slicers

---

## Project Workflow
1. Data import from Excel
2. Data cleaning & transformation
3. KPI creation
4. Dashboard development
5. Supply chain insights visualization

This project helped strengthen my skills in supply chain analytics, KPI reporting, dashboard design, and interactive data visualization using Power BI.
