# Sales Insights Dashboard Project

## Project Overiew

This Project focus on analyzing sales performance  and business trends using SQL, MySQL, and Power BI. The dashboard provides interactive insights into revenue, sales quantity, customers, products, and market performance.

## Tools & Technologies
- SQL
- MySQL
- Power BI
- Excel

---

## Key Features
- Revenue Analysis
- Sales Quantity Analysis
- Revenue Trend Visualization
- Top 5 Customers Analsis
- Top 5 Products Analysis
- Market-wise Revenue & Sales Insights
- Interactive Year & Month Slicers
- Data Cleaning & Data Modeling in Power BI

---

## Project Workflow
1. Data import & preprocessing
2. Data analysis using SQL queries
3. Data cleaning & transformation in Power BI
4. Data modeling & relationship building
5. Interactive dashboard development

This project helped strengthen my skills in data analysis, SQL querying, data visualization, and business intelligence reporting
